# Uber Clone (MERN Stack)

## Tech Stack
- Frontend: React, Vite
- Backend: Node.js, Express, Socket.io
- Database: MongoDB Atlas
- Real-time: WebSockets via Socket.io

## Features
- Signup/Login
- Request rides
- Real-time driver tracking
- Chat between user and driver
